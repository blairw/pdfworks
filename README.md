# pdfworks

For `collate-adf.sh`, you will need `pdftk`. On Mac, you can install the Java ported version (`brew install pdftk-java`).

On Linux, it is probably easier to use `collate-adf-pdf-stapler.sh` with `pdf-stapler`.
